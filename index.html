Active Directory & GPO

Active Directory is a centralized identity and access management platform that provides authentication, authorization, and directory services for users, computers, and resources across enterprise environments.

I work with Active Directory and Group Policy to design and manage secure identity structures, organizational units, and policy-based configurations, enforcing security baselines, access controls, and operational standards across domain-joined systems while ensuring stability, compliance, and scalability.

BigFix Architecture

BigFix is an enterprise endpoint management and compliance platform used to maintain visibility, security, and control across large-scale device environments.

I work with BigFix to manage endpoint posture by enforcing patch compliance, software deployments, configuration baselines, and security remediation, while integrating with Active Directory for identity-aware device management and reporting.

Exchange Hybrid Flow

Exchange Hybrid is a coexistence model that integrates on-premises Microsoft Exchange with Exchange Online, enabling a unified messaging experience across hybrid environments.

I work with Exchange Hybrid to design and operate secure mail flow, directory synchronization, and authentication between on-premises Active Directory, Exchange Server, and Entra ID, ensuring seamless user experience, compliance, and minimal disruption during cloud adoption or migration.

Entra Conditional Access

Entra Conditional Access is a core Zero Trust capability that controls access to applications and data based on identity, device state, location, and risk signals.

I design and manage Conditional Access policies to enforce secure authentication, MFA requirements, compliant device access, and risk-based controls across cloud and hybrid environments, ensuring strong security without impacting user productivity.
